# Merch Vendor Platform

A B2B merchandise vendor platform integrated with Shopify for managing custom apparel (t-shirts, hoodies, polos, hats, tumblers, engraving, etc.).

## Features

- **Customer Storefront**: Browse and purchase pre-configured vendor merchandise
- **Vendor Portal**: Manage products, pricing, and orders
- **Shopify Integration**: Seamless integration with your Shopify store
- **Inquiry System**: New customers can inquire about becoming vendors
- **Order Management**: Track orders from purchase to shipment
- **Product Customization**: Support for various merchandise types and customization options

## Tech Stack

- Node.js + Express (Backend API)
- React (Frontend)
- PostgreSQL (Database)
- Shopify Admin API
- Shopify Storefront API

## Project Structure

```
├── backend/              # Node.js/Express API server
├── frontend/             # React application
├── database/             # Database migrations and schemas
├── config/               # Configuration files
├── public/               # Static assets
└── docs/                 # Documentation
```
